## üíª Installation & Setup

1. **Clone the repository**
   ```bash
   git clone https://github.com/yourusername/ai-event-planner.git
   ```

2. **Install dependencies**
   ```bash
   cd ai-event-planner
   npm install
   ```

3. **Start development server**
   ```bash
   npm run dev
   ```

4. **Build for production**
   ```bash
   npm run build
   ```
